Afin d’exécuter le programme il est nécessaire d’enregistrer le fichier global dans Windows(:C) afin d’assurer le 
bon fonctionnement des chemins des fichiers utilisés .

le fichier 'data prof.txt' correspond aux données des enseignants
le fichier 'test2.txt' correspond aux données des étudiants
le fichier 'file admin.txt' correspond aux données des admin
le fichier 'file abs.txt' correspond aux données des absences

le dossier 'assets' contient les éléments graphiques nécessaire pour créer les espaces tkinter de la plateforme
